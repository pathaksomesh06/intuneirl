<h2 id="what-is-parallels-software"><strong>What is Parallels Software?</strong></h2>
<p>Parallels is a globally recognized software that allows users to run Windows, Linux, and indeed another instance of macOS, on their Mac, without rebooting. The software provides a seamless way to run applications and perform tasks on different operating systems simultaneously.</p>
<p><strong>Preparing for the Installation</strong></p>
<ul><li><strong>Download Parallels Software</strong>: The first step is to get your copy of Parallels Desktop for Mac. You can download it directly from the Parallels website. </li></ul>

<ul><li><strong>Ensure System Requirements</strong>: To create a virtual macOS, your system must meet the minimum requirements set by Parallels. The details of these requirements can also be found on the Parallels website. </li></ul>
<p><strong>Install Parallels on your Mac</strong></p>
<ul><li>Double-click on the downloaded .dmg file, followed by another double-click on "Install Parallels Desktop" within the resulting window. If prompted for admin confirmation, choose "Open" to proceed.</li><li> Following that, please wait while Parallels downloads the required installation data. Once completed, enter your Password and click on "Continue" if prompted to choose between the Trial version. Then, proceed to accept the license agreement and select "OK" for any permission prompts that may appear. Finally, follow any remaining instructions to successfully finish the installation process.</li></ul>

<h2 id="use-the-microsoft-enterprise-sso-plug-in-on-macos-devices"><strong>Use the Microsoft Enterprise SSO Plug-in On macOS Devices</strong></h2>
<h3 id="prerequisites">Prerequisites</h3>
<p>To use the Microsoft Enterprise SSO plug-in on macOS devices:</p>
<ul><li>The device is managed by Intune.</li><li>The device must be running on macOS 10.15 and newer</li><li>The Microsoft Company Portal app must be installed and configured on the device.</li></ul>
<p>In the <a href="https://intune.microsoft.com">Microsoft Intune admin center</a>, create a device configuration profile. This profile includes the settings to configure the SSO app extension on devices.</p>
<ol><li>Sign in to the <a href="https://go.microsoft.com/fwlink/?linkid=2109431">Microsoft Intune admin center</a>.</li><li>Select <strong>Devices</strong> &gt; <strong>Configuration profiles</strong> &gt; <strong>Create profile</strong>.</li><li>Enter the following properties:<ul><li><strong>Platform</strong>: Select <strong>macOS</strong>.</li><li><strong>Profile</strong>: Select <strong>Templates</strong> &gt; <strong>Device features</strong>.</li></ul></li><li>Select <strong>Create</strong>:</li><li>In <strong>Basics</strong>, enter the following properties:<ul><li><strong>Name</strong>: Enter a descriptive name for the policy. Name your policies so you can easily identify them later. For example, a good policy name is <strong>macOS: Microsoft Enterprise SSO plug-in</strong>.</li><li><strong>Description</strong>: Enter a description for the policy. This setting is optional, but recommended.</li><li>Select <strong>Next</strong>.</li><li>In <strong>Configuration settings</strong>, select <strong>Single sign-on app extension</strong>, and configure the following properties:<ul><li><strong>SSO app extension type</strong>: Select <strong>Microsoft Azure AD</strong>:</li></ul></li><li><strong>App bundle ID</strong>: Enter a list of bundle IDs for apps that don't support MSAL <strong>and</strong> are allowed to use SSO. </li><li>Continue creating the profile, and assign the profile to the users or groups that will receive these settings. </li></ul></li></ol>
